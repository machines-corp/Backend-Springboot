# Backend-Springboot

Este repositorio contendrá la API y la conexión a la base de datos del proyecto.  
La aplicación está basada en **Spring Boot** y se conectará a una base de datos relacional (ej: MySQL/PostgreSQL).  

## Próximos pasos
- Configuración de dependencias en **Maven**.
- Creación de entidades y repositorios.
- Implementación de controladores REST.
- Integración con la base de datos mediante JPA/Hibernate.
